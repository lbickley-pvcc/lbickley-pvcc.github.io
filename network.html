Chapter 4 Terms:

Chapter Sections Terms/Definitions
Network Classifications|
  • LAN- a collection of computers in a single building or building complex
  • MAN- a network of intermediate size, such as one spanning a local community
  • WAN- links machines over a greater distance—perhaps in neighboring cities or on opposite sides of the world.
Network models
  • bus network- machines are all connected to a common communication line
  • star network- is a LAN topology in which all nodes aka PCs/workstations/other devices connect to a centeral computer often refered to as a hub
Protocols
  • CSMA/CD Ethernet- based on the Ethernet standards, the right to transmit messages is controlled by the protocol
  • CSMA/CA Wifi- is wireless network policy of trying to avoid collisions rather than trying to detect them
Combining networks
  • repeater- is that wireless networks adopt the policy of trying to avoid collisions rather than trying to detect them
  • bridge- similar to a repearer but is that wireless networks adopt the policy of trying to avoid collisions rather than trying to detect them
  • switch- a bridge with multiple connections, allowing it to connect several buses rather than just two
  • router- The connection between networks to form an internet is handled by devices
  • gateway- The “point” at which one network is linked to an internet
Client/server model
  • p2p: peer-to-peer - popular means of distributing files such as music recordings and motion pictures via the Internet
  • distributed system - consist of software units that execute as processes on different computers
  • cloud computing- uge pools of shared computers on the network can be allocated for use by clients as needed
Internet Addressing
  • Tier 1- ISPs that consist of very high-speed, high-capacity, international WANs
  • Tier 2- ISPs that tend to be more regional in scope and less potent in their capabilities
  • Tier 3- Access to this core is usually provided by an intermediary
IP address
  • ICANN- nonprofit corporation established to coordinate the Internet’s operation
  • DNS- these name servers are used as an Internet-wide directory system
  • TLD: Top-level domain- suffixes on domains examples are .edu, .gov. mil, .net
  • Country-code TLDs - two- letter specific country TLD domains
Internet protocols
  • FTP- an application for listing and copying files across the network implemented
  • HTTP- application layer protocol in the Internet protocol suite model for distributed, collaborative, hypermedia information systems
  • SMTP- a way that two computers on the network may interact when transmitting an email message from one host to the other
World Wide Web
  • URL- each document is given a unique address
  • client-side activities- activities if they are performed by a client (such as a browser)
  • server-side activities- activities if they are performed by a server (such as a webserver)
Internet four-layer software
  • application layer- Prepares message and provides destination address
  • transport layer- Chops message into packets
  • network layer- Assigns intermediate address to each packet
  • link layer- Transfers packet
Internet is:
  • packet-switching network
  • fault tolerant
  • redundant
Cybersecurity
  • malware- malicious softwares
  • virus- software that infects a computer by inserting itself into programs that already reside in the machine
  • worm- an autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
  • Trojan Horse- a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim
  • spyware- software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
  • phishing- a technique of obtaining information explicitly by simply asking for it.
  • DDoS attack- process of overloading a computer with messages.
Protection
  • firewall- A primary prevention technique is to filter traffic passing through a point in the network
  • spam filter- firewalls designed to block unwanted email
  • proxy server- a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server
  • antivirus software- software designged to defend against invasions via network connections
  • cryptography- study of sending and receiving messages securely in the presence of adversaries
  • public key encryption
